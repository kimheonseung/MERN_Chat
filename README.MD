## Chat Demo With MERN Stack
  

#### Example Demo Application Using MERN Stack
  
  
  
## Backend  
  
  
### Stack
- MongoDB
- Express
- Node
  
---
  
### Features
- User Register
- User Login
- User Authentication with JWT
- Chat using Websocket
- Join / Out Chat Room
